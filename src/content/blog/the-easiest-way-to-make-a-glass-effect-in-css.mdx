## 4- Add glassÂ effect

Adding some styling & glass effect.

```css
.box {
  position: relative;
  width: 400px;
  height: 200px;
  background-color: rgba(255, 255, 255, 0.15);
  box-shadow: 0 0 1rem 0 rgba(0, 0, 0, 0.2);
  border-radius: 5px;
  backdrop-filter: blur(15px); /* glass effect */
}
```
